This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

Website part of the UCSD Police Department Log aggregator, run on Next.js and pulls data from Supabase database made from my Scraper repo that scrapes pd logs whenever a new log gets published. This front end allows for multiple view types, search, filters, and sorting functionality.
